Docker Commons Plugin

DockerImageExtractor

org.jenkinsci.plugins.docker.commons.DockerImageExtractor in docker-commons

Identifies the Docker images used by a Jenkins job. Docker-related plugins can use this extension point to integrate with for example Docker HUB web hook support.

Implementations:

  • CloudBees Docker Custom Build Environment Plugin: com.cloudbees.jenkins.plugins.docker_build_env.SelectorDockerImageExtractor (view on GitHub)

  • CloudBees Docker Hub/Registry Notification Plugin: org.jenkinsci.plugins.registry.notification.DockerPullImageBuilder.ImageExtractor (view on GitHub)

  • CloudBees Docker Hub/Registry Notification Plugin: org.jenkinsci.plugins.registry.notification.TriggerImageExtractor (view on GitHub)

  • Docker Pipeline Plugin: org.jenkinsci.plugins.docker.workflow.ImageAction.ExtractorImpl (view on GitHub)
